1.0.2 - 2018-01-04

  • Additions
    • Support for Clang-4.0
    • Support for GCC-7
    • Support for MSVC-2017
    • Fixed support for MSVC-2015 Update 3
    • Core
      • Support for injecting templates/concepts
    • Extensions
      • Extensions are now available using header files
      • Extensible Injector
      • Shared Factory
  • Bug Fixes
    • Implementation class cannot be marked as final
    • Creation with Factory doesn't allow to pass exceptions through
    • Bound policy doesn't work with multiple bindings
    • Problems compiling with _GLIBCXX_DEBUG defined
    • Exception in ctor not propagated
    • Unable to compile with VS 2017 RC, VS 2015 works fine
    • Broken under VS2015 Update 3
  • Contributions
    • Kanstantsin Chernik

1.0.1 - 2016-05-04

  • Breaking Changes
    • Simplified interface for policies
  • Additions
    • Support for Clang-3.9
    • Support for GCC-6
    • Extensions
      • Contextual Bindings
      • Serialize
  • Bug Fixes
    • Link error if di.hpp is included in two cpp-files
    • Creation of empty injector crashes on MSVC 2015, Update 1, x64, Debug

1.0.0 - 2016-02-22

  • Support for MSVC-2015+
  • Support for GCC-5+
  • Support for CMake 2.8+
  • Support for libc++, libstdc++
  • Support for Mac OS
